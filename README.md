# Sales Analytics — Professional Project

This repository contains a professional sales analytics notebook ready for GitHub and portfolio submission.

## Contents
- `Sales_Analytics_Pro.ipynb` — polished, documented notebook
- `data/` — place your dataset here (Advanced_Sales_Report.xlsx)
- `outputs/` — charts and generated reports
- `src/` — helper scripts (optional)
- `requirements.txt` — Python dependencies

## How to run
1. Install dependencies: `pip install -r requirements.txt`
2. Open the notebook in Jupyter or Colab
3. Run cells top-to-bottom
